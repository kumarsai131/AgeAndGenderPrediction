    About

    For the Age prediction model, I used the UTKFace dataset is a large-scale face dataset with a long age span (range from 0 to 116 years old).

    • Dataset has over 20000 images.
    • Used ktrain API to train the model.
    • Ktrain is a lightweight wrapper for keras to train neural networks.

    For the Gender prediction model, I used the IMDB-WIKI dataset, the largest publicly available dataset on the internet.

    • Dataset has over 200000 images.
    • Dataset is split into Train, Validation and Test.
    • Used sequential model of the Keras API.
